# Category SEO descriptions and filter specs

Paste these into **Products → Categories** for each category.  
**Filter specs:** The `filters` field holds references to **Spec** documents. There are no spec documents in the dataset yet. After you create specs (e.g. "Cold Water Rated", "Weight", "Size"), open each category and add the relevant specs to **Filter Specs**. Leave empty until then.
